python安装简记

Posted on 2015-12-02 | Edited on 2017-11-21 | In Python Notes | | Views: | Views:

简介:

本文主要简介python在Windows、Linux(ubuntu、kubuntu)上的安装以及一些简单的设置，适合初学者，高手绕行啊:

目前，使用Python的主要版本2.7和3.0，这两个版本在语法上的差异还较大，但具体的思想异曲同工。

Windows平台上的安装

下载地址：Python官方下载地址
下载主流的版本，下载格式为.exe，下载完成后双击运行，使用默认的安装目录即可。

注意，请点选将Add python.exe to Path

!!! Add python.exe to Path点选will be installed on local hard drive 之后就是傻瓜式的安装过程了

Pycham IDE

Windows平台下推荐使用Pycham IDE完整的Python开发环境。
下载地址：Pycham官方下载地址

主要有专业版和社区版，选择社区版下载安装即可。IDE会自动搜索Python的安装环境，之后再安装相应的插件，如：numpy和matplotlib等。
